# 설치 방법 설명
※ 마인크래프트 계정이 있어야 가능하다.

---

1. PC 상에서 사이드 퀘스트 설치
2. 휴대폰과 기기 연결하기, 개발자 모드로 변경(생략가능(?)) 
3. 오큘러스 기기 키고(초록불 다 들어와있는 지 확인.) 사이드 퀘스트 상에서 제공하는 해당 번호 마이크로 소프트 상에서 적기.
4. 앱 실행 시키고 로그인 하기.
